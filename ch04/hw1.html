HTML 문서에 CSS3 스타일 시트 만드는 방법 3가지
  1. <style></style> 태그에 스타일 시트 작성
  2. sytle 속성에 스타일 시트 작성(태그 뒤에 속성으로 사용)
  3. 스타일 시트를 별도 파일로 작성(<lint> 태그나 @import로 불러서 사용)

CSS3 규칙2 - 스타일 합치기와 오버라이딩
- 태그에 적용 가능한 스타일(아래의 번호가 높을수록 우선수위가 높다)
  1. style 속성에 선언된 스타일
  2. <stytle></style> 태그에 선언된 스타일
  3. 스타일 시트 파일에 선언된 스타일
  4. 브라우저의 디폴트 스타일
- 스타일 합치기(cascading)와 오버라이딩(overriding)이란
태그에 적용되는 모든 스타일이 합쳐지고, 동일한 스타일은 우선 순위가 높은 스타일부터 적용되는 규칙이다

셀렉터(selector) - HTML 태그의 모양을 꾸밀 스타일 시트를 선택하는 기능이다(여러 유형의 셀렉터 존재)

  • 태그 이름 셀렉터 - 태그 이름이 셀렉터로 사용되며 셀렉터와 같은 이름의 모든 태그에 적용된다
  • class 셀렉터 - 점(.)으로 시작하는 이름의 셀렉터이며 HTML 태그의 class 속성으로만 지정 가능하다
  • id 셀렉터 - #으로 시작하는 이름의 셀렉터이며 HTML 태그의 id 속성으로만 지정 가능하다

class 셀렉터와 id 셀렉터 비교
-class 셀렉터
  • 여러 태그를 하나의 그룹으로 묶어 단체로 동일한 CSS 스타일을 적용할 때 적합하다
  • class 속성 값이 같은 태그에 모두 CSS 스타일이 적용된다
  • 태그의 종류에 관계없이 class 셀렉터 활용 가능
-id 셀렉터
  • id 속성의 목적은 각 태그를 유일하게 구분(중복 X)
  • 동일한 id 속성을 가지지 않도록 HTML 파일을 작성하는 것이 바람직하다
  • 자바스크립트 코드에서 id 값을 가진 태그 객체를 찾을 때 문제된다
  • id 셀렉터는 여러 태그 중 특정 태그에만 css 스타일을 적용할 때 적합하다

  • 자손 셀렉터 - 자손 관계인 2개 이상의 태그 나열

CSS3의 박스 모델 - HTML 태그는 사각형 박스로 다루어진다
  • 콘텐츠(content) - HTML 태그의 텍스트나 이미지가 출력되는 부분
  • 패딩(padding) - 콘텐츠를 직접 둘러싸고 있는 내부 여백
  • 테두리(border) - 패딩 외부의 테두리로서, 테두리를 원하는 형태로 그릴 수 있다
  • 여백(margin) - 박스의 맨 바깥 영역이며 테두리 바깥의 공간이다
