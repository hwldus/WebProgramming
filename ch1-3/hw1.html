2주차 1~3장 요약

  • HTML5는 웹페이지를 기술하기 위한 markup 언어이며 차세대 웹 표준이다.
    • 문법
    요소(element) = 시작태그 + 콘텐츠 + 종료 태그 (요소 중에 br, hr, img 등 콘텐츠가 없는 요소도 있다.)
    속성(attribute) = 요소의 성질, 특징을 정의 / 이름 = 값
  • 기존의 HTML과 다르게 문서를 구조화할 수 있다.
  • 이때 시맨틱 태그(문서의 구조와 의미를 전달하는 태그)를 사용한다.
    모양과 구조는 별개이므로 직접 위치, 색, 모양 등을 css3를 이용해 지정해야 한다.
    • 시맨틱 태그 종류
    <header> <section> <article> <nav> <aside> <footer>
  • 여러가지의 태그를 써서 원하는 모습의 페이지를 만들 수 있다.
  • 또한 간단하게 글자 모양, 밑줄 등을 조절할 수 있다.

    • 글자 모양 관련 예시

    글자 모양

    작게 굵게 이태릭체

    글자 밑줄

    하이라이트 취소선 밑줄 아래첨자 위첨자


    위의 예시처럼 <details> 태그를 이용하면 사용자가 핸들을 클릭 해 항목을 숨기거나 볼 수 있다.
    <figure> 태그를 사용하면 책이나 본문 등에 삽입하는 사진, 차트, 등을 하나의 그림으로 표현할 수 있다.
    웹 폼
    웹 페이지에서 사용자 입력을 받는 폼
    ex)로그인, 등록, 검색, 예약, 쇼핑 등
    웹 요소
    폼을 만드는 다양한 태그
    ex)<input>, <textarea>, <select> 등
    폼 태그 <form> 태그로 둘러싸는 모양이다.
    name 속성: 폼의 이름 지정
    action 속성: 폼 데이터를 처리할 웹 서버 응용프로그램 이름, submit 버튼이 눌리면 브라우저는 action에 지정된 응용프로그램 실행을 요청
    method 속성: 폼 데이터를 웹 서버로 전송하는 방식(ex. GET, POST)
