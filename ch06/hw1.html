웹 페이지에서 자바스크립트의 역할

  • 사용자의 입력 및 계산
  • 웹 페이지 내용 및 모양의 동적 제어
  • 브라우저 제어
  • 웹 서버와의 통신
  • 웹 어플리케이션 작성
js코드 위치
  1. HTML 태그의 이벤트 리스너 속성에 작성
    2. - ex) onclick=~
  2. <script></script> 태그에 작성
    3. - head나 body 내 어디든 가능하며 여러 번 삽입할 수 있다
  3. 자바스크립트 파일에 작성
    4. - 확장자 .js파일에 저장한다.(<script> 태그 없이 코드만 저장)
      =><script> 태그의 src 속성으로 파일을 불러 사용한다.
  4. URL부분에 작성
    5. - ex) <a href="javascript:alert("클릭")">클릭</a>

HTML 콘텐츠 출력

- document.write()로 출력


자바스크립트 다이얼로그(프롬프트, 확인, 경고)

ex) prompt(script태그 안에 넣어서 실행하면 됨)
let ret = prompt("이름을 입력하세요");
if(ret == null) {
// 취소 버튼이나 다이얼로그를 닫은 경우
} else if(ret == "") {
//문자열 입력 없이 확인 버튼 누른 경우
} else {
// ret에는 사용자가 입력한 문자열
}

자바스크립트 식별자
- 자바스크립트 프로그램의 변수, 상수(리터럴), 함수의 이름 자바스크립트 문장
- 세미콜론(;) 사용
- 주석문(한 줄: //, 여러 줄: /**/)

데이터 타입

- 숫자(정수, 실수), 논리(참, 거짓), 문자열, 객체 레퍼런스(객체를 가리킴), null
- 자바스크립트에는 문자타입이 없고 다 문자열("a")로 표현

변수

- 프로그램이 실행 중에 데이터를 저장하는 공간
- 변수 선언(이름을 정하고, 저장 공간 할당)
- 자바스크립트에는 변수 타입이 없다
- var키워드(언어가 도입될 때부터 있었고 지금도 사용)
- let키워드(재선언X, 재할당 O)

상수

- const키워드(한번 저장되면 값을 바꿀 수 없다)
- 재선언, 재할당 X, 블록 범위에서만 사용

this
- 지역변수와 전역변수의 이름이 같을 때 this.전역변수 를 사용해 접근(단, let으로 선언된 전역변수는 접근 불가)
리터럴(데이터 값 그 자체)
- 종류: 정수, 실수, 논리, 문자열, 기타(null, NaN)
- 문자열 리터럴: 이중 인용 부호와 단일 인용 부호 모두 사용가능

연산자

- 종류: 산술, 증감, 비트, 시프트, 대입, 비교, 논리, 조건
- 산술 연산자(+, -, *, /, %), 결과는 항상 실수
ex)
- 증감 연산자(++, --), 전위연산자, 후위연산자
- 대입 연산자: 오른쪽 식의 결과를 왼쪽 변수에 대입
- 비교 연산자: 두 값 비교, true, false의 결과를 내는 연산
- 논리 연산자(AND, OR, NOT)
- 조건 연산자(condition ? expT:expF)
ex)


- 비트 연산(비트들끼리 비트 논리 연산, 비트 시프트 연산)
- 문자열 연산자(+): 문자열 연결, 비교 연산자로 문자열 비교도 된다.

비교문

- 종류: if, if-else문, switch문
- if, if-else문: 조건에 따라 비교하기
- switch문: 값에 따라 서로 다른 코드를 실행할 때 적합
case 문의 값은 const 선언된 상수나 리터럴만 가능(변수나 식은 사용 불가)
break문: 만날 때까지 아래로 코드 계속 실행(블록범위)
ex)

반복문

- 종류: for문, while문, do-while문
- for문 ex)

- while문과 do-while문 차이
while문은 조건을 먼저 보고 만족하면 반복하지만 do-while문은 일단 처음에는 무조건 한번 실행한다.
- break문과 continue문 차이
break는 가장 안쪽 반복문 하나만 벗어나고 continue는 반복 코드 실행을 중단 후, 다음 반복으로 점프

함수

- 특정한 일을 처리할 수 있도록 미리 만들어 놓은 코드
- 작업을 위한 데이터를 받아서 일을 처리한 후, 결과물을 되돌려준다.
- 함수의 구성

- 자바스크립트에서 제공하는 전역 함수
- eval(): 이중 인용 부호 무시, parseInt(): 문자열을 숫자로 변환, isNaN(): Not a Number
